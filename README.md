# Snake Difficulty APK

A tiny native Android Snake game where every score increases the difficulty.

## Gameplay

- Swipe to move the snake.
- Eat the red food to score.
- Every food pickup increases speed.
- The game tracks high score during the current install.
- Tap **Restart** after game over.

## Build APK on GitHub

1. Open the repo on GitHub.
2. Go to **Actions**.
3. Run **Build APK** manually, or push a change.
4. Open the completed run.
5. Download the `SnakeDifficulty-debug-apk` artifact.
6. Install `app-debug.apk` on your Android phone.

This is a debug APK for testing, not a Play Store release build.
